像素小人技术对比 Demo

1. Canvas 2D 硬编码

推荐度: ★★★★★
✓ <5KB, 完全可控
✗ 需手写帧数据

2. CSS Sprite

推荐度: ★★★★
✓ 设计师友好
✗ 需要图片资源

3. CSS box-shadow

推荐度: ★★★
✓ 纯CSS, <2KB
✗ 复杂动画难

4. SVG + CSS

推荐度: ★★★
✓ 矢量可缩放
✗ 像素风不适合

5. GIF / APNG

推荐度: ★
需要外部GIF文件
50-200KB
✓ 最简单
✗ 文件大，不可控

6. WebGL / Three.js

推荐度: ★★
需要引入Three.js
150KB+，杀鸡用牛刀
✓ 功能最强大
✗ 过于重型

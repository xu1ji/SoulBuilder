**5阶段时序**（总计18秒）：

| complete | 等待 | idle | I know [skill]. |

## 三、App 包设计（待开发）

### 3.1 职责定义

**App 包是整个应用的"胶水层"**，负责：

1. **路由管理** - 页面导航与参数传递
2. **状态管理** - 全局 Zustand store
3. **页面组装** - 组合各子包的组件
4. **API层** - 与后端通信
5. **布局** - Header、主内容区、响应式

interface AppState {

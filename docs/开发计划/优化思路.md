## 二、核心优化方案

### 2.1 架构升级：从纯前端到前后端分离

```
当前架构（纯前端 Demo）：
┌─────────────────────────────────────┐
│           index.html                │
│    app.js (所有逻辑硬编码)            │
└─────────────────────────────────────┘

目标架构（推荐）：
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   前端 UI     │ ←→  │   后端 API    │ ←→  │   LLM + 搜索  │
│  (现有代码)   │     │  (新增)       │     │   (外部服务)  │
└──────────────┘     └──────────────┘     └──────────────┘
```

**技术选型建议**：

| 层级 | 方案 A（快速 MVP） | 方案 B（长期架构） |
|-----|------------------|------------------|
| **后端** | Vercel Serverless Functions | Next.js API Routes / Bun + Hono |
| **LLM** | Claude API (Anthropic SDK) | 同左 + 支持多模型切换 |
| **搜索** | Tavily API | Tavily + 自建缓存层 |
| **最佳实践库** | JSON 文件 | Supabase (PostgreSQL) |
| **部署** | Vercel | Vercel / 自托管 |

### 2.3 核心模块二：方法论搜索引擎

**设计目标**（参考产品方案第三章）：

```
用户描述岗位 → 识别类型 → 决策树
    │
    ├─ 经典岗位 → 查官方库
    │   ├─ 命中 → 秒级注入
    │   └─ 未命中 → 联网搜索 → 注入
    │
    ├─ 复合岗位 → 拆解子领域 → 分别搜索 → 融合
    │
    └─ 个人分身 → 跳过搜索 → 侧重性格采集
```

```javascript
// backend/methodology-cache.js
import methodologies from '@/data/methodologies';

export function findCachedMethodology(userInput) {
  const input = userInput.toLowerCase();

  for (const [roleId, data] of Object.entries(methodologies)) {
    // 检查别名匹配
    if (data.aliases.some(alias => input.includes(alias.toLowerCase()))) {
      return {
        hit: true,
        data: data.methodology_pack,
        quality: data.quality_score
      };
    }
  }

  return { hit: false };
```

### 2.5 核心模块四：配置文件生成

**5 个文件的生成逻辑**（参考数字生命架构指南）：

```javascript
// backend/generate-files.js

export function generateConfigFiles(portrait) {
  const { profile, soul, knowledge, methodology, protocol } = portrait;

  return {
    'IDENTITY.md': generateIdentity(profile, methodology),
    'SOUL.md': generateSoul(soul),
    'HEARTBEAT.md': generateHeartbeat(methodology, profile),
    'AGENT.md': generateAgent(protocol),
    'USER.md': generateUser(knowledge.employer)

function generateSoul(soul) {
  return `# SOUL.md - Who You Are

${soul.boundaries.map(b => `- ${b}`).join('\n')}

}

function generateHeartbeat(methodology, profile) {
  // 根据岗位匹配 3-5 条心跳任务
```

### 2.6 核心模块五：ZIP 导出修复

**当前 Bug**：`btnExportZip` 错误绑定到 `exportMarkdown`

**修复方案**：

```javascript
// 引入 JSZip 库
import JSZip from 'jszip';
import { saveAs } from 'file-saver';

async function exportZip() {
  const zip = new JSZip();

  // 生成 5 个配置文件
  const files = generateConfigFiles(state.portrait);

  // 添加到 ZIP
  for (const [filename, content] of Object.entries(files)) {
    zip.file(filename, content);
  }

  // 生成并下载
  const blob = await zip.generateAsync({ type: 'blob' });
  saveAs(blob, `${state.roleName}_配置文件包.zip`);
}

// 修复绑定
els.btnExportZip.addEventListener('click', exportZip);
```
